# Component Registry

The DeDevs UI registry contains 40+ production-ready components across multiple domains. These components are designed for AI interfaces, blockchain/Web3 applications, developer portfolios, and general-purpose modern web applications.

## Quick Start

Install any component using the DeDevs UI CLI:

```bash
npx dedevs-ui add [component-name]
```

Install multiple components:

```bash
npx dedevs-ui add button card dialog form
```

## Advanced Features

### Search & Filter
Use the category tabs above to filter components by type. Each component includes:
- **Installation command** - Copy-paste ready CLI commands
- **Dependencies** - Required packages and versions
- **Category badges** - Quick visual categorization
- **Component names** - Clear identification of each component

### Bulk Installation
Install multiple components efficiently:

```bash
# Install a complete form set
npx dedevs-ui add form input label button select

# Install layout components
npx dedevs-ui add card accordion separator tabs

# Install overlay components
npx dedevs-ui add dialog sheet popover tooltip
```

### Domain-Specific Collections

**AI Interface Components:**
```bash
npx dedevs-ui add command progress skeleton toast chart
```

**Web3/Blockchain Components:**
```bash
npx dedevs-ui add button card badge alert progress
```

**Portfolio Components:**
```bash
npx dedevs-ui add card carousel badge button tabs
```

## Component Dependencies

All components are built with:
